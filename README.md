# 🎬 Video Editor Portfolio Website

A modern and fully responsive **portfolio website for a video editor**, built using **HTML, CSS, and JavaScript**.  
This project showcases editing skills, projects, and creative work through an interactive UI with animations and video elements.

## 🚀 Features
- 🎥 **Background Video Homepage** with autoplay (muted)
- 🧭 Smooth **Navigation Bar**
- 👤 **About Section** with personal details
- 🛠️ **Skills Section** highlighting editing tools & expertise
- 📂 **Projects Section** with 12+ videos
- ▶️ Videos with **autoplay & muted sound**
- ⭐ **Client Reviews / Testimonials Section**
- 📞 **Contact Page**
- ✨ Smooth animations & transitions
- 📱 Fully responsive (Mobile + Desktop)
- 🌑 Dark theme UI (modern & professional)

## 🛠️ Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

## 🎯 Purpose
This portfolio is designed to:
- Showcase video editing skills
- Display projects in an engaging way
- Attract clients or job opportunities
- Build a strong online presence
